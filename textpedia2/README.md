# textpedia-api
Repo for the Textpedia API

### CREATE A USER

```
{
  "data": {
    "email": "email",
    "phoneNumber": "phoneNumber"
  }
}
```

Endpoint: /submit

Method: POST

### CONFIRM TOKEN

```
{
  "data": {
    "jwt": "jwt",
    "token": "auth token"
  }
}
```

Endpoint: /confirm

Method: POST
